## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/books-app.git
```

Go to the project directory

```bash
  cd books-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
